The steel module code can be viewed at https://gist.github.com/ronkok/d42b0efdc66dc4f6135fee3b8d22a83e

It exposes four functions:

Ps(section, Fy, kLx, kLy) returns LRFD column strength
Pas(section, Fy, kLx, kLy) returns service level allowable column strength
Ms(section, Fy, Lb, Cb, axis) returns LRFD beam strength
Mas(section, Fy, Lb, Cb, axis) returns service level allowable beam strength

All four functions must be invoked in unit-aware mode.
