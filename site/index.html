This week’s example:

Beam Analysis

Hurmet introduces a new feature, beam analysis. It can be found at https://hurmet.app/ce/BeamAnalysis.html.

On that page, you can specify a beam and do an analysis. Then you can copy the results with a Ctrl-C and paste them into a Hurmet page, like this one. Here’s an example:

Input

E: 29,000 MPa
Section: W8X31
I: 110 in⁴
Beam: f 12.5′ p 3.125′ —

wD = -0.031 kips/ft
wL = -0.1 -0.2 kips/ft; 0′ -1′
PL = -2 kips at 3′

Load combinations: service
Patterned live loads: yes
