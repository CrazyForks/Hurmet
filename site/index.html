This is editable text. You can focus it and start typing. You can open a calculation cell by clicking on it. All calculations will update if you press Shift-Enter to close a calculation cell. You can create a calculation cell by pressing Alt-C.

Breaking Change

Henceforward, a delimited matrix literal must be preceded by a “\” character.

\(1, 2; 3, 4)

\[a; b; c]

I do this because otherwise it was impossible to write a math interval, as in or .

I will try hard to avoid any further breaking changes in Hurmet syntax.
