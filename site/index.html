Getting calculations right

This is editable text. You can focus it and start typing. A calculation cell will open if you click on it. All calculations will update if you press Enter to close a calculation cell.

Change

To close a calculation cell, press Enter.
To create a newline inside a cell, press Shift+Enter.

Breaking Change

A literal data frame must now be delimited by double backticks, not single backticks.

Single backticks have a new purpose. They will enable a calculation to return rich text.

Suppose that a cell should test a condition. If the condition is met, it should return .
If the condition is not met, it should return .

Until now, the result of a calculation could not write an italic or a subscript, as in . Now it can. Just write `M_u`.
