Hurmet Units

The values in columns length through money are exponents that Hurmet uses to check unit compatibility. For instance, a liter is a measure of volume, or length3, so the table’s “length” column shows a 3 for liters.

The currency exchange rates in this table are only placeholders. Hurmet actually uses data from the European Central Bank. This data is updated once per week. You can define your own, more accurate exchange rates to override the default rates.

If you are curious about some of the more unusual units, such as “survey foot” or “nautical mile”, I recommend Russ Rowlett’s dictionary of units of measurement.
